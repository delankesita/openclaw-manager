# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-03-30

### Added
- **Refactored Architecture**: Clean separation of concerns with dedicated modules
- **Dashboard UI**: Modern webview panel with instance cards and real-time status
- **Tree View**: Sidebar panel showing instance hierarchy with context menus
- **Health Monitoring**: Automatic health checks with CPU/memory tracking
- **Instance Management**: Full lifecycle control (create, start, stop, restart, clone, delete)
- **Config & Logs**: Quick access to instance configuration and log files
- **Port Management**: Automatic port detection and collision avoidance
- **Auto-start**: Configurable auto-start of instances on VS Code launch

### Improved
- Better error handling and user feedback
- Cleaner codebase with TypeScript strict mode
- Improved instance isolation and state management

## [0.1.0] - 2026-03-30

### Added
- Initial release
- Basic instance management
- Dashboard and tree view
- Health monitoring
